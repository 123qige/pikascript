# usage

#### 1. Run the bsp/xxx/pikascript/pikaPackage.exe to get source code of pikascript core and packages.

#### 2. Run the bsp/xxx/pikascript/rust-msc-latest-win10.exe to pre-compile the packages of pikascript.

#### 3. Enjoy.
